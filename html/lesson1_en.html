Problem Background

Carbon dioxide (CO2) emissions are one of the main causes of global warming. Automobiles are significant sources of emissions. If we can accurately predict a vehicle's CO2 emissions based on its attributes (such as engine size, fuel consumption, etc.), we can help governments formulate policies and assist consumers in choosing environmentally friendly vehicle models.

Objective: Build a model that takes vehicle characteristics as input and outputs predicted CO2 values.

Initial Data Exploration

We used the public dataset FuelConsumptionCo2.csv from the Government of Canada. It mainly includes the following fields:

  • ENGINESIZE: Engine displacement (L)
  • CYLINDERS: Number of cylinders
  • FUELCONSUMPTION_COMB: Combined fuel consumption (L/100km) - Key feature
  • CO2EMISSIONS: CO2 emissions (g/km) - Target we aim to predict
