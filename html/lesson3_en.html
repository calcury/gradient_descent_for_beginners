Loss Function

Since we cannot directly solve the equations, we define a loss function to measure how "bad" the model is.

For linear regression y = kx + b, we use Mean Squared Error (MSE):

$$L(k, b) = \frac{1}{n} \sum_{i=1}^{n} (y_i - (kx_i + b))^2$$

Our goal now becomes: Find a set of (k, b) values that minimize L(k, b).

Visualization: Loss Function Surface

If we take k and b as the x and y axes, and the loss value L as the height (z-axis), we get a bowl-shaped surface.

The lowest point represents the optimal k and b values.
