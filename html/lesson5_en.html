Challenge: Curse of Dimensionality

When we have 10, 50, or even more features:

  • We cannot visualize the loss function surface (humans cannot imagine 50-dimensional space).
  • Full-batch Gradient Descent (GD) requires computing gradients on all data for each update, making it too slow.

Solutions:

  1. SGD (Stochastic Gradient Descent): Updates using only one sample at a time - fast but noisy.
  2. GD/SGD with Learning Rate Decay: Learning rate decreases dynamically with training epochs for more stable convergence.
  3. Adam Optimizer: Combines momentum and adaptive learning rate for faster and more stable convergence.
  4. PCA (Principal Component Analysis): Dimensionality reduction that preserves key information, reduces computation, and lowers initial loss.
